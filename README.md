# arduino-projects
Arduino-based practice projects

## Projects

### Ultrasonic Distance Sensor with LED Warning
- Measures distance using HC-SR04
- Turns on LED when object is closer than 20 cm
- Uses Serial Monitor for distance output

Concepts used:
- Digital I/O
- Timing and pulse measurement
- Sensor integration

### Kullanılan bağlantılar (HC-SR04)
- VCC → 5V
- GND → GND
- TRIG → D9
- ECHO → D10
- LED → D13 (Arduino üzerindeki dahili LED)
